# Акча табууну үйрөтүүчү Telegram бот

Бул бот:
- Тил тандоону колдойт (Кыргызча / Орусча)
- Боттун ичине кирүү үчүн 3 каналга катталууну милдеттүү кылат
- Ар бир кирүүдө (жана ар бир бөлүмгө өтөрдө) катталуусун кайра текшерет —
  эгер колдонуучу каналдан чыгып кетсе, "катталган эмессиз" деп кайра эскертет
- Курстар бөлүмүн камтыйт (сабактар тизмеси → толук түшүндүрмө текст + YouTube видео шилтемеси)
- Админ панелин камтыйт: бардык колдонуучуларга текст/фото/видео жөнөтүү (рассылка), колдонуучулар санын көрүү
- 24/7 колдоо баскычын камтыйт

## 1. Орнотуу

```bash
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## 2. Жөндөө (config.py)

`config.py` файлын ачып төмөнкүлөрдү толтуруңуз:

1. **BOT_TOKEN** — Telegram'да `@BotFather`'га жазып, "/newbot" менен жаңы бот түзүп,
   ал берген токенди жабыштырыңыз.

2. **ADMIN_IDS** — Сиздин (же башка админдердин) Telegram ID'иңиз.
   ID'иңизди билбесеңиз, `@userinfobot` ботуна жазыңыз — ал ID'иңизди көрсөтөт.
   Бир нече админ кылса болот: `ADMIN_IDS = [111111, 222222]`

3. **CHANNELS** — Катталуу милдеттүү болгон 3 канал.
   - `chat_id`: `@kanal_username` түрүндө же сандык ID (`-1001234567890`)
   - `url`: колдонуучу басып өтүүчү шилтеме
   - **ЭСКЕРТҮҮ:** бот ошол 3 каналдын **админи** болушу керек, болбосо
     Telegram API катталганын текшерүүгө уруксат бербейт!

4. **SUPPORT_CONTACT** — колдоо кызматынын @username же телефону.

## 3. Курстардын мазмунун өзгөртүү (courses_data.py)

`courses_data.py` файлында `COURSES` сөздүгүндө `ky` жана `ru` тилдери үчүн
сабактар тизмеси турат. Ар бир сабакта:
- `title` — сабактын аталышы
- `text` — толук түшүндүрмө текст
- `youtube` — ошол сабакка тиешелүү YouTube видеонун шилтемеси

Жаңы сабак кошуу үчүн эки тилге да (ky жана ru) бирдей орунда кошуңуз
(тизмелердин узундугу бирдей болушу керек).

## 4. Ботту иштетүү

```bash
python bot.py
```

Бот туура жөндөлсө, терминалда "Бот иштеп баштады..." деген билдирүү чыгат.

## 5. Кантип иштейт (кыскача)

- `/start` → тил тандоо (биринчи жолу) → 3 каналга катталганы текшерилет
- Катталбаган болсо → каналдардын шилтемелери + "✅ Текшерүү" баскычы чыгат
- Катталган болсо → башкы меню: 📚 Курстар, 🆘 Колдоо, 🌐 Тил, (админ болсо) 🛠 Админ панель
- Курстар бөлүмүнө же башка бөлүмгө кирээрде да катталуу кайра текшерилет
- Админ панелинде:
  - **📢 Бардыгына билдирүү жөнөтүү** — баскычты басып, кийинки билдирүүңүздү
    (текст, фото, видео — каалаганы) жөнөтсөңүз, ал бардык колдонуучуларга
    автоматтык түрдө жөнөтүлөт. Жокко чыгаруу: `/cancel`
  - **📊 Колдонуучулар саны** — ботту колдонгондордун жалпы санын көрсөтөт

## 6. Серверге туруктуу иштетүү (көрсөтмө)

24/7 иштеши үчүн VPS'ке (мисалы systemd кызматы катары) же
`screen`/`tmux`/`pm2`/`supervisor` сыяктуу процесс менеджер аркылуу
фондо иштетиңиз. Мисал, systemd:

```ini
[Unit]
Description=MoneyBot Telegram Bot
After=network.target

[Service]
WorkingDirectory=/path/to/moneybot
ExecStart=/path/to/moneybot/venv/bin/python bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```

## Файлдардын структурасы

```
moneybot/
├── bot.py              # Ботту иштетүүчү негизги файл
├── config.py            # Токен, админдер, каналдар, контакт
├── database.py           # SQLite база (колдонуучулар, тилдер)
├── locales.py            # Бардык тексттер (ky/ru)
├── courses_data.py        # Сабактардын мазмуну
├── keyboards.py           # Inline баскычтар
├── subscription.py         # Катталууну текшерүү логикасы
├── requirements.txt
└── handlers/
    ├── start.py           # /start, тил тандоо, башкы меню
    ├── courses.py          # Сабактар бөлүмү
    ├── support.py          # 24/7 колдоо
    └── admin.py            # Админ панели, рассылка
```
